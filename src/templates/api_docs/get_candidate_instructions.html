REST API METHOD: GET Instructions

Retrieving a list of posted candidates that are associated with a GW event.

Usage

/api/v1/candidate

Notes

This endpoint is for getting a list of candidates that are associated with a GW event.

The parameters my be passed in via a json argument in the request or as URL querystring arguments (i.e. /api/v1/candidate?param1=value1...)

Parameters

  • api_token (str; verified account API Token)
  • graceid (str; graceid associated with GW event)
  • id (int; id for single candidate)
  • ids (list[int]; list of ids for multiple candidates)
  • userid (int; user id associated with api_token that posted candidates)
  • submitted_date_after (str(datetime); filters candidates created after or on a specified datetime)
  • submitted_date_before (str(datetime); filters candidates created before or on a specified datetime)
  • discovery_date_after (str(datetime); filters candidates discovered after or on a specified datetime)
  • discovery_date_after (str(datetime); filters candidates discovered after or on a specified datetime)
  • discovery_magnitude_gt (float; filters candidates with magnitude greater than (or equal) specified value)
  • discovery_magnitude_lt (float; filters candidates with magnitude lesser than (or equal) specified value)
  • associated_galaxy (str; filters candidates that are submitted with an associated galaxy name)
  • associated_galaxy_redshift_gt (float; filters candidates with associated_galaxy_redshift greater than (or equal) specified value)
  • associated_galaxy_redshift_lt (float; filters candidates with associated_galaxy_redshift lesser than (or equal) specified value)
  • associated_galaxy_distance_gt (float; filters candidates with associated_galaxy_distance greater than (or equal) specified value)
  • associated_galaxy_distance_lt (float; filters candidates with associated_galaxy_distance lesser than (or equal) specified value)
