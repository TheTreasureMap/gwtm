REST API METHOD: POST Instructions

USAGE

/api/v0/update_pointings

/api/v1/update_pointings

NOTES

  • Will simply cancel all planned pointings'
  • You can only cancel pointings associated with your API_TOKEN. You cannot cancel other peoples planned pointings

Parameters

  • api_token=abcdefghijkl
  • status=status1
    • must be updated from planned to cancelled
  • id=id1
  • ids=[id1, id2, id3...]
