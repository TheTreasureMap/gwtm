REST API METHOD: POST Instructions

USAGE

/api/v0/cancel_all

/api/v1/cancel_all

NOTES

  • Will simply cancel all instruments' planned pointings' statuses for a given graceid
  • You can only cancel pointings associated with your API_TOKEN

Parameters

  • api_token=abcdefghijkl
  • graceid=graceid1
  • instrumentid=instrumentid1
